My Personal Blog

This is the first project I did along with the website itself. I followed a course about making a website on Django framework. And this blog application was the example. So, this project helped me learnt:

  • Django project structure, like views, urls, and setting, admin modules to manage the website.
  • Django model class and how it's an abstraction of data base interface int order to mangage data inplace rather than write another piece of data base code to manage data.

Please also note that this blog app, like the webgame app, mainly serves as an exercise for me to get the idea of web development rather than an actual functioning app, but I have plans to improve it to a well-designed blog module like so many other examples online. The first step is to impletment a index panel that could sort or retrieve entries based on date or most recent updates. And also a coment section.
