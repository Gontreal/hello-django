This is the final exercise of the book above that I read to learn python, it's implemented on web.py framework in the book, but I move it to Django, as a start point to get to know this framework. I learnt:

  • Basic python OOP analysis and design, implementing unit test cases both local and in the Django framework.

Please note that this game is still very primitive, just for the purpose of getting the basic ideas of web development.
